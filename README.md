<h2>🚀 Quick Start</h2>
<p>Want to run this site locally? It's simple:</p>
<pre>
  1. Clone the repository
     <code>git clone https://github.com/Lidorpahima/portfolio.git</code>
  
  2. Open index.html in your browser
     <code>cd portfolio && open index.html</code>
  
  3. That's it! No build process or dependencies required
</pre>
